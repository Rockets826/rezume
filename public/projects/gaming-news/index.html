О проекте

Веб-платформа для публикации и чтения игровых новостей. Пользователи могут просматривать статьи по категориям, искать контент и читать материалы на адаптивном интерфейсе.

Архитектура

  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: PHP с MVC-подходом
  • База данных: PostgreSQL
  • CRUD для статей, категорий и авторов

Какие задачи решались

  • Публикация и редактирование новостей
  • Категоризация контента
  • Поиск по заголовкам и описаниям
  • Адаптивная вёрстка для mobile/desktop

Сложности

  • Проектирование структуры БД для контента
  • Организация backend-логики без фреймворка
  • Баланс между скоростью разработки и чистотой кода

Чему научился

Закрепил навыки backend на PHP, работы с PostgreSQL и построения полноценного CRUD-приложения с нуля.
