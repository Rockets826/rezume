Campus Navigation System

3D-навигация по кампусу университета (ЛГУ)

Информационная система навигации по кампусу университета. Проект включает 3D-визуализацию зданий на WebGL, модели из Blender, поиск помещений и построение маршрутов внутри корпусов. Данные хранятся в PostgreSQL.

Ключевые возможности

  • 3D-визуализация корпусов и макета района (GLB-модели)
  • Интерактивная сцена на Three.js / WebGL
  • Поиск помещений по названию
  • Построение маршрутов внутри здания
  • Работа с PostgreSQL для хранения данных о помещениях

Технологии

PHP JavaScript Three.js PostgreSQL Blender WebGL
